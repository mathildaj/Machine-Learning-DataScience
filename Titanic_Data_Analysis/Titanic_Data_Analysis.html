# Because Age has such a large range in values, let's create a new column called AgeBin to divide the Age into groups

# Create a function to create the bin group
def age_bin(passenger):
    if passenger['Age']> 0 and passenger['Age']<= 10:
        return '0-10'
    elif passenger['Age']>10 and passenger['Age']<= 20:
        return '11-20'
    elif passenger['Age']>20 and passenger['Age']<= 30:
        return '21-30'
    elif passenger['Age']>30 and passenger['Age']<= 40:
        return '31-40'
    elif passenger['Age']>40 and passenger['Age']<= 50:
        return '41-50'
    elif passenger['Age']>50 and passenger['Age']<= 60:
        return '51-60'
    elif passenger['Age']>60 and passenger['Age']<= 70:
        return '61-70'
    elif passenger['Age']>70 and passenger['Age']<= 80:
        return '71-80'

I see that Pclass has the strongest negative correlation with survivorship: -0.338481. It means the higher the class number was (which means the "the lower the class"), the less likely the passenger was going to survice.

Age also has a negative correlation with with survivorship, but not as strong: -0.077221. So, the older the passenger was, the lessly likely he/she was going to survive.

Conclusion:

  1. Sex had a significant impact on the surviorship. Females were almost three times more likely to survive.
  2. Children under 10 were more likely to survive.
  3. From 1 and 2, it seems that "women and young children come first" likely happened on the ship.
  4. 1st class passengers were much more likely to survive.
  5. Having family did increase the odds of survival.
  6. The fare price between 101-200 had the highest odds of survival.
  7. Passengers embarked from Cherbourg had the highest odds of survival.

Limitation of the analysis:

  1. Age column has 177 values missing out of 891 records. The missing values may impact the age distribution of the survivalship. I chose to drop the missing age values when computing the analysis, as I didn't want to fill them with 0s. 0 in the place of missing age could skew the age distribution to the younger group.

  2. Embarked column also has 2 missing values. I ddin't drop these 2. Because Embarked is a catogorical variable, and I didn't think 2 out of 891 missing would make a big impact.

  3. According to Wikipedia, Titanic had around 1317 passengers (actually, the exact number of passengers are unknown), and the dataset used in this analysis only has 891 passengers. So this is a sampe out of the population. My analysis didn't include statistical testing. The conclusion was based on more of my own observations, and not statistically significant.

  4. Other varibles not included in this dataset could have been useful in the analysis, such as: the ability to swim, race/ethnic group, profession, etc.
